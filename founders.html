This website has been developed by us as part of our academic project, focusing on web development and artificial intelligence. As the founders and creators, we aimed to design an interactive and user-friendly platform that combines modern web design with the power of AI technology.

Through this project, we not only learned the fundamentals of web development — including layout design, responsive structure, and coding — but also explored how artificial intelligence can enhance user experience. One of the key features of this project is a small AI-powered chatbot, trained to assist users and provide relevant information efficiently.
